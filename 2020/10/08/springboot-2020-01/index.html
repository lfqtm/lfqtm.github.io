springboot-2020-01

Posted on 2020-10-08 Edited on 2020-10-09 In 尚硅谷

Spring Boot是由Pivotal团队提供的框架,目的是用来简化Spring应用的初始搭建及开发过程

引言

Spring Boot(微框架) = Spring MVC(控制器) + Spring(项目管理)

1. SpringBoot特点

  • 创建独立的Spring应用
  • 嵌入Tomcat,无需部署war文件
  • 简化maven配置
  • 自动配置Spring,没有XML配置

2. 约定大于配置

快速构建hello world

maven构建,gradle构建, spring initializer构建…参看官网

1. 约定大于配置

  • 项目结构
  • pom.xml
  • application.yml - 主配置
  • Applicaton - 主入口类

2. 运行方式

  • 运行主入口程序
  • 配置jvm参数

  • maven打包通过java -jar xxx.jar运行

java -jar SpringBoot_01-1.0-SNAPSHOT.jar
